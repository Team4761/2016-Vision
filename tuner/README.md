Vision Tuner Program
====================
Graphical program that allows you to move sliders around so you can visually see what RGB filters will work.

![Screenshot of Tuner Program](http://i.imgur.com/vNhsqBj.png)

Compiling
---------
```
make
```

Requirements
------------
 * OpenCV

Running
-------
 * `./tuner [PATH-TO-IMAGE]`
 * `./tuner --help`
